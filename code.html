cleaning: turn data into a csv

CSV tends to be easier to work with and more malleable by programming languages than Excel.

Excel/Sheets

Download as CSV.

csvkit

## This will save the CSV file in the same directory as where the Excel file exists.

in2csv excel_filename.xlsx > csv_filename.csv

R

## R will overwrite a file if the name is already in use.

write.csv(name_of_dataframe,'name_of_csv.csv')

Python

## This converts a Pandas dataframe into CSV.

import pandas as pd

df = pd.DataFrame({'name': ['Emi', 'Railey'], 'color': ['black', 'gray'], 'age': ['4', '2']})
